- # Fitur Admin
    - Autentikasi Admin: Login, logout, proteksi halaman dengan middleware.
    - Dashboard Admin: Menampilkan statistik buku, pengguna, peminjaman, dan grafik aktivitas.

    - Pencarian dan Detail Buku: Daftar buku, pencarian, detail buku, dan tombol Pinjam Buku.
    - Peminjaman Buku: Ajukan pinjaman, lihat status, batalkan sebelum disetujui.
    - Profil Pengguna: Edit profil, ubah password, upload foto profil.
